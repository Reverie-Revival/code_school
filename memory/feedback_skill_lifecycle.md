---
name: feedback-skill-lifecycle
description: Rule to register new skills immediately when created for Code School
metadata:
  type: feedback
---

Any new project-specific skill created for Code School must be added to [[feedback-skill-reminders]] immediately, not at some later cleanup pass.

**Why:** A skill that exists but isn't registered is a skill that won't get remembered or used consistently.

**How to apply:** Update the registry in the same session the skill is created.
